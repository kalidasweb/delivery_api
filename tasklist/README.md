# Task List

Used to collect a Token for a registered User.

**URL** : `/api/new_order/`

**Method** : `POST`

**Auth required** : Yes

**Headers required**:

 ```x-client-data: <client_token>```

 ```x-client-date: <Current_Time_String>```

```Authorization: <auth_token>```
